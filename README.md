# Customer Spending Analytics

## Project Overview

This project analyzes ecommerce sales data to uncover insights about revenue, profitability, regional performance, product categories, and discount impact.

The analysis was performed using **Python (Pandas)** for data cleaning and analysis and **Power BI** for interactive dashboard visualization.

---

## Key Business Questions Answered

* What is the total sales and profit performance?
* Which regions generate the highest revenue?
* Which product categories are most profitable?
* How do discounts affect profitability?
* What are the monthly sales trends?

---

## Tools Used

* Python
* Pandas
* Power BI

---

## Key Insights

**Revenue Performance**

* Total sales exceeded ₹533M across 5000 orders.

**Regional Analysis**

* North region generated the highest revenue.

**Category Profitability**

* Furniture had the highest profit margin (15.35%).
* Beauty category showed the lowest margin.

**Discount Impact**

* Small discounts (~5%) slightly improved profitability.
* High discounts (15–20%) reduced average profit.

**Seasonality**

* Sales peaked around May and December.
